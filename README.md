Regenmengenmesser
=================

Aufgabe 1: Das Regenwasser soll in einer Zisterne gesammelt werden. Erstellen Sie einen Flow, der in Abhängigkeit von der Regenmenge den Zulauf zur Zisterne regelt. Folgende Vorgaben müssen beachtet werden: - Die Regenmenge muss >= 10ml sein, damit die Zisterne befüllt wird. - Bevor der Zulauf geöffnet bzw. geschlossen wird, muss überprüft werden, ob er nicht bereits geöffnet bzw. geschlossen ist. - Muss der Zulauf geöffnet werden, dann soll diese Aktion erst nach einer Pause von 2 Sekunden erfolgen. - Wurde der Zulauf geöffnet oder geschlossen, dann soll dies über eine Textausgabe im Debug- Bereich angezeigt werden. Tipps: - Benutzen Sie für Status des Zulaufs eine Flow-Variable. - Testen Sie den Ablauf, in dem Sie für die Regenmenge Zufallswerte benutzen.
